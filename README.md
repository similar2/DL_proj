## 如何添加仓库：
```shell
# 首次添加仓库：
git init  #  初始化仓库
git remote add origin git@gitee.com:xbxzka/digital-logic-project.git   # 添加远程仓库
git pull --set-upstream origin master # 拉取仓库内容

# 添加仓库后：
git push # 推送内容
git pull # 拉取内容
```
